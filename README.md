# 🌍 Países de Interés

A responsive, visually modern webpage showcasing three countries — **Brasil, Noruega y Japón** — built with HTML and CSS.

## 📋 Description

This page presents key information about three fascinating countries through an interactive card-based layout. Each card includes a country image, flag badge, short description, and a link to its Wikipedia page.

## 🚀 Features

- Dark theme UI with country-coded accent colors
- Responsive card grid (adapts to mobile)
- Smooth hover animations and fade-in on load
- Flag badge overlaid on each country image
- Graceful image fallbacks with emoji placeholders
